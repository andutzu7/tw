Descrierea proiectului

RUnDa(Romanian Unemployment Data) este un instrument Web de vizualizare si comparare multi-criteriala a datelor publice referitoare la somajul din Romania pe ultimele 12 luni.

Structura

In continuare va vom prezenta tehnologiile folosite :

HTML

HyperText Markup Language (HTML) este un limbaj de marcare utilizat pentru crearea paginilor Web ce pot fi afișate într-un browser (sau navigator)

CSS

CSS (Cascading Style Sheets) este un standard pentru formatarea elementelor unui document HTML.

Baze de Date

MySQL este un sistem de gestiune a bazelor de date relaționale, aflat sub Licența Publică Generală GNU, si este cel mai popular SGBD open-source la ora actuala.

JavaScript

JavaScript (JS) este un limbaj de programare orientat obiect bazat pe conceptul prototipurilor. Este folosit mai ales pentru introducerea unor funcționalități în paginile web, codul JavaScript din aceste pagini fiind rulat de către browser.

Python

Python este un limbaj de programare dinamic multi-paradigmă, si multifunctional care permite atat programarea orientata pe obiecte cat și programarea imperativă, funcțională sau procedurală.
